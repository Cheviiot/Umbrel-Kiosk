## ⚡ Установка

Выполните команду на устройстве с UmbrelOS:

```bash
curl -fsSL https://raw.githubusercontent.com/Cheviiot/UmbrelOS-Rus/main/scripts/umbrel-russify.sh | sudo bash
```

## 🎉 Готово

После установки откройте **Настройки → Язык → Русский**
